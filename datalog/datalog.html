Data Log Generation

This web interface enables OpenDosimeter users to connect their devices via USB and visualize the stored dose log. Users can also download the data as a CSV file for further analysis. Please note that this page is still under development, so the output may appear unusual when the "Print Log" button is pressed. If this occurs, simply press the button again. The x-axis displays the time since the last dose reset, with gaps in the data representing power cycles (when the device has been turned off and on).
